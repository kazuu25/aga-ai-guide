AIの全体像

AIというと難しく聞こえますが、使うのは2つの道具だけ。 話しかけて動いてくれる「相棒」と、なんでもためておく「倉庫」。 この2つを一度つないでおけば、あとは思ったことを伝えるだけで、どんどん前に進みます。 まずは全体像を見て、そのあと「はじめの設定」を、いっしょに1つずつやっていきましょう。

Claude Code（クロード）
話しかけると、調べたり・書いたり・作ったりしてくれる「動く相棒」。あなたが指示を出す相手です。
Obsidian（オブシディアン）
メモや資料をためておく「自分専用の記憶の倉庫」。ここに、どんどん蓄えていきます。
この1回の「つなぐ」設定が、今回いっしょにやることです。
1スクショで伝える
2Obsidianをつなぐ
3フォルダにためる
4作業を引き出す
